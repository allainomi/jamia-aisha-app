## ایپ کی خصوصیات
- مکمل **آف لائن** چلتی ہے  
- **الارم** کے ساتھ نماز کا وقت  
- **ہجری تاریخ** خود بخود اپ ڈیٹ  
- **ٹائم ٹیبل** کلاسز کے لیے  
- **Expo Go** میں فوراً چلے گی

1. **Expo Go** ایپ ڈاؤن لوڈ کریں

3. ایپ فوراً چلے گی!
