# جامعہ عائشہ صدیقہ لبنات الاسلامیہ - موبائل ایپ

یہ ایپ جامعہ عائشہ صدیقہ لبنات الاسلامیہ مچھی پول، مانسہرہ کی آفیشل ایپ ہے۔ طالبات کے لئے ٹائم ٹیبل، ہدایات، داخلہ، امتحانات، ڈیجیٹل واچ، اسلامی کیلنڈر، نوٹیفکیشنز وغیرہ شامل ہیں۔

## فیچرز
- آف لائن استعمال
- ہجری تاریخ اور ڈیجیٹل واچ
- روزانہ الارم (صبح 7:30 بجے)
- ٹائم ٹیبل اور شیڈول
- داخلہ فارم
- فاضلات کی فہرست
